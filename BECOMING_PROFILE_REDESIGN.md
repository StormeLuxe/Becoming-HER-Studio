# BECOMING PROFILE UX REDESIGN

## Current behavior:

When a user logs in, they see:
- Dashboard
- Becoming Profile

However, when they click Becoming Profile, the page is locked behind the Creator plan.

This creates a poor onboarding experience because users cannot establish their identity before seeing the platform.

---

## NEW REQUIREMENT

**The Becoming Profile should be FREE for every user.**

It is the onboarding foundation of Becoming HER Studio.

Every user should be able to complete it immediately after signup.

**Do NOT lock Becoming Profile behind a paid subscription.**

---

## BECOMING PROFILE REDESIGN

Instead of a single profile screen, create a guided onboarding experience with cards or tiles.

Users should see multiple sections with descriptions.
